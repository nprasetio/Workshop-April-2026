# 📊 Metabase — Panduan Workshop

**Sesi 3 & 4 | Total: 120 menit**

Metabase adalah platform **Business Intelligence (BI)** open source yang memungkinkan siapa pun — tidak harus paham SQL — membuat visualisasi data dan dashboard interaktif langsung dari database.

---

## 🎯 Tujuan Pembelajaran

Setelah sesi ini, peserta mampu:
- Memahami konsep BI dan kegunaan Metabase
- Menghubungkan Metabase ke database Neon PostgreSQL
- Menggunakan Query Builder untuk eksplorasi data tanpa SQL
- Menulis SQL query di Metabase SQL Editor
- Menyimpan hasil query sebagai Question
- Membuat dashboard interaktif dengan multiple chart
- Menambahkan filter dan drill-down ke dashboard
- Membagikan dashboard ke orang lain

---
